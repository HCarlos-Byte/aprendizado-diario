Internet - Como surgiu?

Podemos dizer que o primeiro meio de comunicação foi o Telegrafo lá em 1958 sendo ultilizado o método de código morse. Avançando 10 anos em 1969 a ARPANET lançou um sistema de comunicação em pacotes, enviados um a um e que permitia multiplos destinos. A primeira conexão foi feita em 29/10/1969 entre a univerdidade da california e o instituto de stanford sendo envia a plavra "LOGIN", mas só chegando "LO" sendo assim a primeira falha na comunicação, podemos dizer que a sigla "LO" tambem foi a primeira palavra a circular pela internet. Com a união de Vinton Cerf e Robert Kahn criaram o protocolo TCP/IP inplantando isso na ARPANET criando assim micro-redes como (MILNET, NSFNET, NASA, entre outras). Em 1989, Tim Berners-Lee viu a oportunidade de unir hipertexto com TCP/IP e criar a World Wide Web (WWW) O primeiro navegador da internet, publicando a primeira página Web em 06 de Agosto de 1991 clique aqui para conferir

Client x Server

Basicamente o client é quem consome e o server é quem fornece. O Client quando entra em um site com uma conexão estavel de Internet solicita os arquivos para o Server e ele retorna para o Client os arquivos solicitados. O client tem que ter uma conexão com internet já o server não necessariamente precisa dessa conexão já que ele pode ser um servidor local. Podendedo existir servidores de Arquivos, Segurança, Streaming, Web, E-mail. Sistema operacional para servidores são totalmente diferente dos que usamos nos dispositivos tendo como sitemas (Windows Server, Ubuntu, CentOS, Debian, Fedora, Oracle) e tambem temos os servidores Web com (Apache, NNginx). A comunicação entre Sistema Operacional x Sevidores Web ocorre da seguinte maneira quando o Sistema operacional identifica uma requisição de um client e ele manda o Servidor Web devolver a resposta com o HTML, CSS, Javascript, Imagens, Texto... Tudo isso ocorre com um Hardware que pode ser até mesmo um PC de casa mas obviamente esse servidor será limitado a grandes tarefas.

Funcionamento das TAG'S

Tags são palavras ou códigos entre colchetes angulares, como "<>". Elas são usadas para identificar elementos específicos em um documento HTML. A maioria das tags HTML tem duas partes: uma tag de abertura e uma tag de fechamento. A tag de abertura é usada para iniciar o elemento e é seguida pelo nome da tag, como "<>" para um parágrafo. A tag de fechamento é semelhante, mas inclui uma barra antes do nome da tag, como "/". As tags de fechamento indicam onde o elemento termina. O navegador interpreta as tags HTML ao carregar uma página e renderiza o conteúdo de acordo com as instruções fornecidas pelas tags. Isso inclui a formatação do texto, a exibição de imagens, a criação de links clicáveis e a organização da estrutura da página.
